# DateUtil - 日期处理

[返回文档导航](../00-文档导航.md) | [工具类速查表](3.7-工具类速查表.md)

---

DateUtil 提供日期格式化、解析、计算功能。

## 预定义格式

```java
DateUtil.YEAR                          // yyyy
DateUtil.MONTH                         // yyyy-MM
DateUtil.DATE                          // yyyy-MM-dd
DateUtil.TIME                          // HH:mm:ss
DateUtil.DATETIME                      // yyyy-MM-dd HH:mm:ss
DateUtil.DATETIME_MINI                 // yyyyMMddHHmmss
```

## 主要方法

```java
// 当前时间
Date now = DateUtil.now();

// 格式化
String str = DateUtil.format(date, "yyyy-MM-dd");
String str = DateUtil.formatNow("yyyy-MM-dd");      // 格式化当前时间
String str = DateUtil.DATE.format(new Date());

// 获取日期字符串
String str = DateUtil.getDateStr(date);              // yyyy-MM-dd
String str = DateUtil.getDateStr();                  // 当前日期字符串
String str = DateUtil.getDateTimeStr(date);          // yyyy-MM-dd HH:mm:ss
String str = DateUtil.getDateTimeStr();              // 当前日期时间字符串

// 解析（智能识别多种格式）
Date date = DateUtil.getDate("2024-01-01 12:00:00");
Date date = DateUtil.getDate("2024-01-01");
Date date = DateUtil.getDate("2024/01/01", "yyyy/MM/dd");
Date date = DateUtil.getDate(localDateTime);
Date date = DateUtil.getDate(localDate);

// LocalDateTime 解析
LocalDateTime ldt = DateUtil.getLocalDateTime("2024-01-01 12:00:00");
LocalDateTime ldt = DateUtil.getLocalDateTime(date);
LocalDateTime ldt = DateUtil.getLocalDateTime("2024-01-01", "yyyy-MM-dd");

// UTC 处理
Date date = DateUtil.parseUTCDate("2024-01-01T12:00:00.000Z");
LocalDateTime ldt = DateUtil.parseUTCLocalDateTime("2024-01-01T12:00:00.000Z");

// 按格式解析（支持 TemporalQuery）
LocalDateTime ldt = DateUtil.parse("2024-01-01", "yyyy-MM-dd", LocalDateTime::from);

// 获取格式化器
DateFormat formatter = DateUtil.formatter("yyyy-MM-dd");

// 时间戳（秒）
long ts = DateUtil.dateline();
```

## 智能解析

自动识别格式：`yyyy-MM-dd HH:mm:ss`、`yyyy-MM-dd`、`yyyy-MM`、`HH:mm`、`HH:mm:ss`、UTC 格式。

解析失败返回 null，不会抛异常。
