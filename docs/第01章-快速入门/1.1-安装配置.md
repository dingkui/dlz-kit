## Maven 依赖

在 `pom.xml` 中添加：

```xml
<dependency>
    <groupId>top.dlzio</groupId>
    <artifactId>dlz-kit</artifactId>
    <version>6.6.3</version>
</dependency>
```

最新版本查看：[Maven Central](https://search.maven.org/artifact/top.dlzio/dlz-kit)

---

## Gradle 依赖

```gradle
dependencies {
    implementation 'top.dlzio:dlz-kit:6.6.4'
}
```

Kotlin DSL：

```kotlin
dependencies {
    implementation("top.dlzio:dlz-kit:6.6.4")
}
```

---

## 依赖说明

dlz-kit 只有一个外部依赖：jackson-databind（2.13.0+）。

大多数 Spring Boot 项目默认已包含 Jackson，无需额外配置。若遇版本冲突，可排除：

```xml
<dependency>
    <groupId>top.dlzio</groupId>
    <artifactId>dlz-kit</artifactId>
    <version>6.6.4</version>
    <exclusions>
        <exclusion>
            <groupId>com.fasterxml.jackson.core</groupId>
            <artifactId>jackson-databind</artifactId>
        </exclusion>
    </exclusions>
</dependency>
```

## 验证安装

```java
import com.dlz-kit.json.JSONMap;

public class InstallTest {
    public static void main(String[] args) {
        JSONMap map = new JSONMap("name", "张三", "age", 25);
        System.out.println(map.getStr("name"));  // 张三
        System.out.println(map.getInt("age"));   // 25
    }
}
```

## 常见问题

**Q: 项目没有 Jackson 怎么办？**
A: 添加 dlz-kit 会自动引入 Jackson 依赖。

**Q: 支持哪些 JDK？**
A: JDK 8、11、17、21。

**Q: 需要配置文件吗？**
A: 不需要，开箱即用。
